# ExodusTweaks — Playbooks para Windows

Playbooks da comunidade AME com suporte ao Windows 10 por kelvenapk

## O que é?

Coleção de playbooks `.apbx` que otimizam o Windows removendo bloatware, desativando telemetria e maximizando performance. Todos modificados para suportar **Windows 10** (Build 19041+).

## Como Usar

1. Baixe o [AME Wizard Beta](https://github.com/Ameliorated-LLC/trusted-uninstaller-cli/releases/tag/0.8.4)
2. Escolha e baixe um playbook
3. Desative o Windows Defender temporariamente
4. Execute o AME Beta e carregue o `.apbx`

## Suporte

- **Windows 10** Build 19041+ (21H2/22H2)
- **Windows 11** suportado
